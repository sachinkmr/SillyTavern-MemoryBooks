# Phase 2 — Graded perception: shell + content memory entries — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`). **This plan spans TWO repos** — Group A in SillyTavern-StateTracker (NEW branch), Group B in SillyTavern-MemoryBooks (branch `two-plane-memory`). Execute Group A first (it stamps the fields Group B reads).

**Goal:** When a directed/private event (whisper, DM, in-room text) happens in front of bystanders, emit **two** memory entries — a templated **SHELL** ("Sachin whispered something to Shilpa", gated to the bystanders who were present) + the **CONTENT** (gated to the participants). Bystanders learn *that* it happened, never the content. Un-skips acceptance U6/U7/U8/E4/E5/E7/N1/N4/N10 (+ E1 no-shell-for-thought).

**Architecture (decided 2026-06-21):** shell generation is **StateTracker-stamp-time** (not STMB-only). StateTracker adds `present_cast + type + from + to` to the channel stamp on directed messages (`content_audience` = the existing narrowed `audience`). STMB reads those at memory time and emits the shell (deterministic template, no LLM) gated to `present_cast − content_audience`. **Memory shells only** — the live `applyVisibilityForDraft` shell (bystanders react in-scene) is explicitly deferred. Both behind their existing flags (StateTracker `channelVisibility`; STMB `twoPlaneMemory`, default off).

**Tech Stack:** StateTracker: `node --import ./tests/register-stubs.mjs --test` (tests in `tests/*.test.js`). STMB: `node --test tests/`. Pure ES modules. STMB builds on Phase 1b's `computePlane1Segments`.

## Global Constraints

- **Two repos, two test runners.** StateTracker: `npm test` (= `node --import ./tests/register-stubs.mjs --test`). STMB: `node --test tests/`. Don't cross them.
- **StateTracker flag-off / back-compat:** the directed fields (`type/present_cast/from/to`) are stamped ONLY when `channelType !== 'room'` (i.e., a directed/narrowed message). Normal (room) stamps keep today's exact shape `{id, audience, remote?, reality?}`. `out` is the single exit wrapper — add the fields there, gated.
- **STMB flag-off byte-identity:** shell emission runs ONLY inside the existing `if (isTwoPlane())` two-plane branches; flag-off behavior unchanged.
- **Shell trigger:** emit a shell for a segment iff its messages carry `present_cast` (directed) AND `present_cast ⊋ content_audience` (a bystander exists). If `present_cast == content_audience` → NO shell (E4/N10). A single-perceiver segment is already dropped by 1b → thoughts get no shell (E1).
- **No LLM for shells.** Shell `content` is a deterministic template from `{type, from, to, remote}`. `suggestedKeys: []` (retrieval is by `characterFilter` only).
- **characterFilter = bystanders**, as AVATAR BASENAMES via `buildEntryCharacterFilter` (Phase 0/1a helper) — disjoint from the content entry's gate (participants). Fail-closed N9 applies (unresolved → matches nobody).
- **Channel-typed shell text:** whisper names the target ("X whispered to Y"); text/DM names the target only if the target is co-present-and-not-remote, else "X was texting someone" (U8 hides a remote target).
- **No content leak (N1/N4):** the content entry stays gated to `content_audience` only — never widened by the shell.

export function computeAudience({ here = [], remote = [], authorName = null, isUser = false, userName, directiveTargets = null, canonicalize = null, alwaysWitness = [], reality = null, channelType = CHANNEL_ROOM_ID }) {
    const cn = typeof canonicalize === 'function' ? (s) => lc(canonicalize(s)) : lc;
    const u = cn(userName);
    const H = here.map(cn), R = remote.map(cn);
    const W = (Array.isArray(alwaysWitness) ? alwaysWitness : []).map(cn);
    const unrealTag = isUnreal(reality) ? String(reality).toLowerCase() : null;
    // Phase 2: directed-channel shell metadata (stamped only when channelType is directed).
    const from = isUser ? u : cn(authorName);
    const to = Array.isArray(directiveTargets) ? uniq(directiveTargets.map(cn)) : null;
    const presentCast = uniq([u, ...H, ...R, ...W]);
    const out = ({ id, audience }) => {
        const aud = uniq([...audience, ...W]);
        const o = { id, audience: aud };
        const rem = mergeStampRemote(aud, R);
        if (rem) o.remote = rem;
        if (unrealTag) o.reality = unrealTag;
        if (channelType && channelType !== CHANNEL_ROOM_ID) {        // directed → shell metadata
            o.type = channelType;                                    // 'whisper' | 'text'
            o.present_cast = presentCast;                            // superset of a narrowed audience
            o.from = from;
            if (to && to.length) o.to = to;
        }
        return o;
    };

```javascript
/** shell.js — PURE graded-perception shell helpers (Phase 2). No SillyTavern imports. */
import { buildEntryCharacterFilter } from './memoryEntry.js';

/** Read the directed-channel stamp from a segment's first stamped message (joins via cm.id). */
export function directedMetaForSegment(seg, chat) {
    for (const cm of (seg?.filteredScene?.messages || [])) {
        const ch = chat?.[cm.id]?.extra?.channel;
        if (ch && Array.isArray(ch.present_cast)) {
            return {
                present_cast: ch.present_cast.map(n => String(n).toLowerCase()),
                type: ch.type || 'room',
                from: String(ch.from || '').toLowerCase(),
                to: (ch.to || []).map(n => String(n).toLowerCase()),
                remote: (ch.remote || []).map(n => String(n).toLowerCase()),
            };
        }
    }
    return null;                                                      // not a directed segment
}

function displayName(token, rosterRows) {
    const r = (rosterRows || []).find(x => String(x.name).toLowerCase() === token);
    return r ? r.name : token;
}

/** Build the shell entry for bystanders, or null when there are none (E4). */
export function buildShellEntry(meta, contentAudience, rosterRows, opts = {}) {
    if (!meta) return null;
    const aud = (contentAudience || []).map(n => String(n).toLowerCase());
    const bystanders = meta.present_cast.filter(n => !aud.includes(n));
    if (!bystanders.length) return null;                             // E4/N10: no excluded observer
    const characterFilter = buildEntryCharacterFilter(bystanders, rosterRows);
    const fromName = displayName(meta.from, rosterRows);
    const toPresentNamed = meta.to.filter(t => !meta.remote.includes(t));     // present, non-remote targets
    let content;
    if (meta.type === 'whisper') {
        const tgt = meta.to.map(t => displayName(t, rosterRows)).join(' and ') || 'someone';
        content = `[Private exchange] ${fromName} whispered something to ${tgt}.`;
    } else { // 'text' / 'dm'
        if (toPresentNamed.length) {
            content = `[Private exchange] ${fromName} was texting ${toPresentNamed.map(t => displayName(t, rosterRows)).join(' and ')}.`;
        } else {
            content = `[Private exchange] ${fromName} was texting someone.`;   // U8: remote/absent target hidden
        }
    }
    return { content, suggestedKeys: [], characterFilter, shell: true, from: meta.from, to: meta.to };
}
```

## Self-Review

- **Coverage:** StateTracker stamps `type/present_cast/from/to` (A1) at whisper + directive sites (A2). STMB reads them (`directedMetaForSegment`), builds the templated shell gated to bystanders (`buildShellEntry`, B1), marks it (`STMB_shell`, B2), and emits it in both seams (B3). Acceptance: U6 (whisper names target), U7 (text), U8 (remote target hidden → "someone", N4), E4/N10 (no bystander → no shell), E1 (thought = single-perceiver → dropped by 1b → no shell), N1 (content entry gated to participants only). E5 (two simultaneous channels) → two independent directed segments → two shells, each gated to the right bystander; E7 (partial overhearing) → the partial overhearer isn't in `content_audience` → bystander → gets the shell, never content (2 tiers).
- **Decisions honored:** StateTracker-stamp-time locus; memory-shells-only (live `applyVisibilityForDraft` shell DEFERRED — documented).
- **Flag-off / back-compat:** StateTracker room stamps unchanged (directed fields gated on `channelType !== 'room'`); STMB shell emission only inside `if (isTwoPlane())`.
- **No-leak:** shell `characterFilter` (bystanders) is disjoint from the content entry's (participants); shell `content` is a template that never contains the message; remote/absent target name hidden (U8/N4).
- **Deferred:** live whisper shell (applyVisibilityForDraft); spoken-`to:` vs whisper nuance (treated as 'whisper' type); shell rollup behavior (Phase 3). The queued-path directed-stamp readability (Step 3) — prefer stashing `directedMeta` in the payload if live-chat read is ambiguous at execution.
- **Cross-repo order:** Group A must land before Group B's live behavior works (B1's unit tests use synthetic stamps and pass independently; B3's live emission needs A's stamps).
